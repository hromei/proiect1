Limbajul HTML

HTML(HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html îl găsiți în imaginea următoare care se află la adresa https://commons.wikimedia.org/wiki/File:HTML_logo.png

Imaginea are atributul height de valoare 147.

Tag-urile html cele mai utilizate în realizarea tabelelor sunt:

table definește un tabel
tr un rând într-un tabel
td o celulă într-un tabel

Observație: Pentru a pune un contur unui tabel se poate(dar nu se recomandă)să se pună atributul border="1". La validare va da eroarea: The border attribute on the table element is obsolete.



Formularele se pot crea cu ajutorul tagului form care poate contine tag-urile: input, textarea, label, select, option, button.

Pentru mai multe detalii consultați cursul sau tutorialul HTML Forms and Inputs de la adresa "http://www.w3schools.com/html/html_forms.asp"

Pentru a crea liste se pot folosi tag-urile:

  1. i. ul - listă neordonată
  2. ii. ol - listă ordonată
  3. iii. li - item din listă
  4. iv. dl - listă cu descrieri
  5. v. dt - termen dintr-o listă cu descrieri
  6. vi. dd - descrierea dintr-o listă cu descrieri



Pentru a insera un document HTML în altul se poate folosi tag-ul iframe.

Astfel, exemplul de mai jos inserează conținutul de la adresa https://html.spec.whatwg.org/multipage/
